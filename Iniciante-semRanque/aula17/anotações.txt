Funções para utilizar no array

push para adicionar no final do array:
array.push('Parametro inserido')

Unshift para adicionar no final do array:
array.unshift('Parametro inserido')

pop para REMOVER final do array
array.pop()

posso REMOVER  o valor que sera REMOVIDO do array
'Const removido = array.pop()'

pop para REMOVER do INICIO do array. 
Esse altera todos os indices do array porque remove do inicio.
array.shift()
